STAR BROWSER

Star Browser is a program that browses through the systems, planets, moons and alien races
created by Star Generator. He uses all the datas stored in *.pln, *.mon, *.sun, *.nam, *.cmt
and *.aln.

When executing Star Browser, you must specify a sector data file.
By clicking on the tab, you can view planet, moon, system or alien datas.
To go to a specific planet, race or system, enter its Id in the edit box next the button Go and press
the button.

To get a logfile in ASCII or in html of the system currently browsed, press the button Logfile
and choose the file format.

To browse through the different records use the buttons + or - .

To go to the first record, press the button F, to go the last, press the buton L.

All the datas are read only, except the box Name that could be edited.
Some notes could be written for the sector by clicking the Notes button.

You could save a single system in an another sector file by clicking the Save button
in the Star system display.
Colony, and empire infos aren't saved.

You could print the complete system infos by clicking the Print button.

You could turn off or on the graphics by checking the graphics box. 

Pictures

All the pictures are stored in the pictures directory in jpg format, if you want you could
change and make better ones (I've been a little hurried to finish Star 1.6).
