With this utility you could easily (I hope) create star systems with any astral bodies you want in it.

Quick beginning guide:

Step 1:

First Open an existing sector or create a new blank sector (File > New Sector). Click on Utilities > Star Editor.

Step 2:
Enter the system coordinates in the X, Y and Z boxes.

Step 3:

Choose the spectral type, class and distance (in AU) to the Primary star of all the stars in the system.
If you want a single star system, put None for Secondary and Tertiary stars.
A spectral class should be between 0 and 9. If you entered an invalid number, it will automatically set the value to 0.

Step 4:
When this will be done it's time to create planets around these stars.
If you want the program to create automatically these planets click on the Auto System button.
When done, you can create a new system by going to Step 2 again, or ending the program by going clicking the Close button.
If you want to manually enter the planet datas, click on the Create system button. Go to Step 5.

Step 5:
After clicking on the Create system button, you'll see the Insert Planet buttons be enabled (depending of the number of stars in the system).
Click on the first Inset planet button, it will open a form that will let you create a planet for the Primary Star. A Planet Editor window will appear.
You can choose between  Quick Edit or  Manually Edit by checking the option you want.
If you chose the Quick Edit go to Step 6.
If you chose the Manually Edit go to Step 7.

Step 6:

The Quick Edit needs only to enter the Orbit radius in AU and the planet type you want. Some planet types need a specific Orbit Radius value, an Icy planet can't be nearer to a star than a Terran one for example.
When done, click on the Create planet button.
Go to Step 8.

Step 7:

In the Manual Edit choice you must enter the main planet characteristics.
Check the Molten Core box if the planet core is solid (Earth, Mars...), and unchecked it for gas giants and icy balls.
Check the Auto Unusual box if you want to automatically generate planet unusual characteristics, or unchecked it if you want to manually choose them. (Use the Ctrl+<left click> to choose multiple characteristics).
Check the Auto Satellite function if you want the program to create the planet satellites. If not, unchecked it. Note that when you click on the Insert Moon button for the first time, the program will create and save the planet, and after that any characteristics modification other than creating a new satellite will not be taken into account.
If the Auto Unusual box is unchecked and there is Intelligent Lifeform on the planet, a new form will be displayed for creating this alien race.

If Ask for creation is checked a window will appear asking you if you want to create the planet or not when clicking on Create Planet.

Step 8:
When the planet editing is done, you 'll return to the main Star Editor form, you can insert another planet or finish the editing of this system by clicking on Done.
You can create a new system by going to Step 2 again,  or end the program by clicking on the Close
button.

 

Alien Edit Form


If you cancel the form or close it without clicking on Create Race a window will appear asking you if you want to generate automatically the alien race. If you click on No, the planet will be created without Intelligent Lifeform inhabiting its surface.
