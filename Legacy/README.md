# Legacy Star Generator / StarWin Reference

This folder preserves the original Star Generator / StarWin application and source materials used as reference for Slip Map Tool work.

All credit for Star Generator / StarWin belongs to its original author:

**Aina Rasolomalala**

The bundled application metadata identifies the program as:

- Product: `Star Generator`
- File description: `2.0 beta2`
- File version: `2.0.0.1`
- Company name: `Stargen Software`
- Copyright: `Aina Rasolomalala`

## Original Author Notice

The original documentation in `starwin v2 compiled app/docs/stardoc1.htm` includes this notice from Aina Rasolomalala:

> Star Generator is a freeware and its source code is freely available. You could develope and distribute freely, programs or addons based on Star Generator provided that these derivated works are free of charge and that you include a text referencing Star Generator and my name. If you created such derivated works, please email-me (I'm always happy to know that there's someone who likes Star Generator) and I'll put a link on my homepage. If you wish to distribute derivated works for a fee or charge, please contact me before. All these to say that I put a lot of hours on this program.

The original documentation then lists:

- `Aina Rasolomalala`
- `aina@geocities.com`
- `http://www.geocities.com/TimesSquare/2691/`

The original GeoCities site is no longer live, but archived mirrors of the Star Generator pages exist, including:

- `https://www.oocities.org/timessquare/2691/stargen.html`
- `https://web.archive.org/web/20040916201003id_/http://www.lemoneyes-radio.com/stargen/`

## Author Contact

Aina Rasolomalala's original notice asks that derivative works include a reference to Star Generator and the author's name, and asks creators to email him so he can know about the work and potentially link to it from his homepage.

The available contact information for Aina Rasolomalala appears to be out of date. The original GeoCities email address and homepage are no longer active, and no current contact information has been found. This repository is preserving the attribution and notice above in good faith, and the maintainer would welcome current contact information for the original author so Aina Rasolomalala can be informed about this preservation and migration work.

## Folder Contents

- `starwin v2 compiled app/` contains the compiled Windows application, bundled documentation, sample data, images, logs, and support DLLs.
- `StarWin2OriginalSource/` contains the original Delphi/Pascal source files and related project artifacts as provided for reference.

## Attribution

This repository does not claim authorship of Star Generator / StarWin. The files are included here so the Slip Map Tool project can study, migrate, and interoperate with legacy Star Generator data formats while retaining clear credit to Aina Rasolomalala.
